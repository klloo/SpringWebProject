# SpringWebProject

#### 프로젝트 개요
학습 목적으로 진행한 간단한 웹 게시판 제작 프로젝트

#### 개발 환경
- **IDE** : 이클립스<br>
- **framework** : Spring<br>
- **DB** : Oracle (Amazon RDS)<br>
- **Language** : Java, JSP

#### 구현 기능
- 게시판 CRUD
- 회원가입 / 로그인 
- 게시물 좋아요 / 댓글 
- 게시물 작성에 SummerNote Editor 적용
- 공공api를 사용한 미세먼지 현황 조회
